【建水】排水构筑物设计

地漏

设置场所及要求：

  1. 厕所、盥洗室及其他需经常从地面排水的场所。高级宾馆客房卫生间和有洁净要求的场所，在业主同意时可不设。
  2. 地漏宜设置在易溅水的卫生器具如浴盆、拖布池、小便器(槽)、洗脸盆附近地面的最低处。
  3. 地漏顶标高应低于地面5 - 10mm，地面应以0.01的坡度坡向地漏。
  4. 住宅套设洗衣机专用地漏，排水管道不得接入室内雨水管道。
  5. 公共食堂、公共厨房和公共浴室等排水宜设置网框式地漏口。
  6. 严禁采用钟罩(扣碗)式地漏。
  7. 地湿的分类及适用场所。

选择地漏的要求：

  1. 应能排除地面积水、冲洗水，当地漏可同时排除卫生器具的排水时，不得有冒溢现象。
  2. 具有较好的自清能力，或有容易清渣的构造。
  3. 地漏的水封深度不得小于50mm。
  4. 对于排水水温要求较高的场所，可采用工程塑料聚碳酸酯材质或金属材质的地漏。

检查口

检查口为带有可开启检查盖的配件，装设在排水立管及较长水平管段上，可作检查和双向清通管道之用。

设置要求：

  1. 铸铁排水立管上检查口之间的距离不宜大于10m;塑料排水立管宜每六层设。
  2. 在最低层和设有卫生器具的二层以上建筑物的最高层必须设置检查口;通气立管汇合时，必须在该层设置检查口。
  3. 当立管水平拐弯或有乙字管时，在该层立管拐弯处和乙字管的上部应设检查口。
  4. 生活污、废水横管的直线管段上检查口之间的最大距离应符合下表的规定。
  5. 立管上检查口的设置高度，从地面至检查口中心宜为$1. 0m$，并应高于该层卫生器具上边缘$0.15m$。
  6. 埋地横管上的检查口应设在砖砌的井内。

清扫口

清扫口装设在排水横管上，用于单向清通排水管道的维修口。

设置要求：

1、在连接2个及2个以止的大便器或3个及3个以上的卫生器具的铸铁排水横管上，宜设置清扫口。

2、采用塑料排水管道时，在连接4个及以上的大便器的污水横管上宜设置清扫口。

3、在水流偏转角大于45°的排水横管上，应设清扫口(或检查口)。

4、在排水横管上宜将清扫口设置在楼板或地坪上。

5、设于排水管起点的清扫口与墙面的距离不得小于0. 2m。排水管起始端设置堵头代替清扫口时，堵头与墙面应有不小于0. 4m。

6、生活污、废水横管的直线管段上清扫口之间的最大距离应符合表的规定。

化粪池

​ 是一种利用沉淀和厌氧发酵原理去除生活污水中悬浮有机物的最初级处理构筑物。

砖砌化粪池图集号：02S701 砖砌化粪池 [排水处理构筑物]

设置条件：

  1. 当城镇没有污水处理厂时，生活粪便水应经化粪池处理后排出。
  2. 城镇污水处理厂规划建设滞后于建成生活小区。
  3. 一些大、中城市因排水管网较长，为防止粪便淤积堵塞。（还要考虑老城区管道管径偏小以及老化）
  4. 城市管网为合流制排水系统，生活粪便水应先经化粪池处理后排入合流制排水系统。
  5. 当地环保部门要求所有污水排入市政排水管网前必须要经（上海就不需要）。

化粪池计算：

化粪池容积在计算出来之后，需要进行区域划分，在区域划分的过程中，要灵活处理，很小可能会遇到可以完全均分的情况，这个时候就要进行化粪池的再分配（化粪池最大体积为$100M^3$）。

一般规定：

  1. 含有大量油脂的废水不得排入化粪池,以防影响化粪池的腐化效果。
  2. 化粪池应设在室外。化粪池外壁距建筑物外墙不宣小于5.0m。化粪池距离地下水取水构筑物不得小于30m。化粪池宜设置在接户管的下游端,便于机动车清搁的位置。
  3. 当受条件限制化粪池设置于建筑物内时,应采取通气防臭、防爆等措施和具备清掏的条件。
  4. 化粪池应根据每日排水量、地形、交通、污泥清掏和排水排放条件等因素综合考虑分散或集中设置。
  5. 当进入化粪池的污水量小于或等于$10m^3/d$,应选用双格化粪池,当进入化粪池的污水量大于$10m^3/d$，应选用三格化粪。

构造要求

  1. 矩形化粪池的长度与深度、宽度的比例应按污水中悬浮物的沉降条件和积存数量,以水力计算确定。但深度（水面至池底）不得小于1.3m,宽度不得小于0.75m,长度不得小于1.0m。圆形化粪池直径不得小于1.0m。
  2. 采用双格化粪池时,第一格的容量宜为计算总容量的75%；采用三格化粪池时，第一格的容量宜为计算总容量的60%，第二格和第三格各宜为计算总容量的20%。
  3. 化粪池格与格之间、池与连接井之间应设通气孔洞。
  4. 化粪池池壁和池底,应防止渗漏。
  5. 化粪池顶板上应设有人孔和盖板。

注意点：

选型依据：容积、是否覆土、是否有地下水、顶面是否过车。

化粪池均设通气管。管材为DN100钢塑管(钢管内、外喷塑)

化粪池进、出管管材:IDN≤200m,为排水铸铁管；DN>200mm,为给水铸铁管。

隔油池

设置范围

职工食堂、营业餐厅、酒店及公共餐饮厨房排水系统。

肉类、食品加工排水系统。

含有少量的汽油、媒油、柴油及其他工业用油的污水系统。

计算

  1. 隔油池污水流量按设计秒流量计算；
  2. 含食用油污水在池内的流速不得大于$0.005M/s$；
  3. 含食用油污水在池内的停留时间宜为2~10min；
  4. 人工除油的隔油池内存油部分的容积,不得小于该池有效。

有效容积：$V=Q_{max}·60·t$

式中：

  1. $V$：有效容积（M）；
  2. $Q_{max}$：污水最大秒流量（$M^3/s$）；
  3. $t$：污水在池内停留时间（min），含食用油污水停留时间为2~10min，含矿物油污水停留时间为10min。

对于餐厅，污水最大设计秒流量按下述标准选用：

  1. 营业餐厅$20L/（人·餐）$，工作时间$12h/d$，$K=2.0$；
  2. 职工餐厅$15L/（人·餐）$，工作时间$9h/d$，$K=2.0$。

污水泵房

泵房设置要求

  1. 排水泵房应设在有良好通风的地下室或底层单独的房间内,并靠近集水池。
  2. 不得设在对卫生环境有特殊要求的生产厂房和公共建筑内,不得设在有安静和防振要求的房间邻近和下面。
  3. 排水泵房的位置应使室内排水管道和水泵出水管尽量简洁,并考虑维修检测的方便。

污水泵设计：

设计流量

  1. 按生活排水设计秒流量计算。
  2. 当有排水量调节时,可按生活排水最大小时流量计算。
  3. 消防电梯集水池内的排水泵流量不小于$10L/s$。
  4. 当集水池接纳水池溢流水、泄空水时,应按水池溢流量、泄流量与排入集水池的其它排水量中大者选择水泵机组。

水泵扬程

水泵扬程按提升高度、管道损失计算确定后,再附加一定的流出水头,宣采用$0.02\sim0.03MPa$。排水泵吸水管和出水管流速不应小于$0.7M/s$,并不宜大于$2.0M/s$。

污水泵选择

建筑物内使用的污水泵类型有：潜水排污泵、液下排水泵、立式污水泵和卧式污水泵。

  1. 建筑物内场地较小,排水量不大,采用潜水排污泵和液下排水泵。
  2. 在排水水质或水温对电机有危害的场所,宣采用液下排污泵立式污水泵和卧式污水泵。
  3. 当排水中夹有大块杂物时,宜采用带有粉碎装置潜水排污泵。

污水泵为一备一用,平时交替使用。

被提升的污、废水温度不超过40°,PH值为6~9。

安装要求

  1. 当提升带有较大杂质的污、废水时,不同集水池内的潜水排污泵出水管不应合并排出;当提升一般废水时,可按实际情况考虑不同集水池的潜水排污泵出水管合并排出。
  2. 污水泵宜设置排水管单独排至室外,排水管的横管段应有坡度坡向出口。两台或所台以上的水泵共用一条出水管时,应在每台水泵出水管上装设阀门和止回阀。
  3. 小型潜水排污泵可以根据国家标准设计图集08S305 《小型潜水排污泵选用及安装》选用。
  4. 小型潜水排污泵的安装方式选用：
安装方式 适用场所 优缺点
软管连接移动式 电动机功率$N≤5.5KW$及排出管管径$DN≤80mm$，较清洁污（废）水的排放 安装方式结构简单，造价较低；但检修、维护较不方便
硬管连接固定式 电动机功率$N≤5.5KW$及排出管管径$DN≤80mm$，较清洁污（废）水的排放 安装方式结构简单，造价较低；但检修、维护较不方便

什么是自动耦合装置？

一般指潜水排污泵，因为潜水排污泵安装在液面以下，水质比较差，人无法直接进到集水坑下面进行维修，为了解决维修问题而做的水泵接口快装装置叫自动耦合装置。

自动耦合本身分为4部分，一部分和水泵相接叫滑板，一部分安装在积水坑内叫底座，一部分安装在集水坑口上端固定座，一部分是滑轨引导水泵安装到位和提升到坑外方便维修，整个一套装置就是自动耦合装置。

主要作用：方便拆装、维护。

集水池设计

设置要求：

  1. 生活污水集水池应与生活给水贮水池保持10m以上的距离。
  2. 集水池宜设在地下室最低层卫生间、淋浴间的底板下或邻近位置。
  3. 地下车库坡道处的集水井应尽量靠近坡道尽头处;车库地面排水集水池应设在使排水管、沟尽量居中的地方。
  4. 地下厨房集水坑则设在厨房近位置。
  5. 消防电梯井集水池应设在电梯邻近处,但不应直接设在电梯井内,池底低于电梯井底不小于0.7M。

有效容积：

  1. 不宜小于最大ー台水泵5min的出水量,且水泵在一小时内启动次数不宜超过6次；
  2. 地下室淋浴间按淋浴器100%时使用的秒流量或小时流量来计算；
  3. 消防电梯井集水池的有效容积不得小于2.0m。

构造要求：

  1. 地下室生活污水集水池的池盖应密闭并应设通气管,可与建筑物内的通气管相连通。
  2. 生活废水集水池的池盖宜密闭并设通气管。当采用敞开式生活废水集水池时,应设强制通风装置,换气次数不小于15次/h。
  3. 地下车库坡道处集水井,车库、泵房、空调机房等处地面水的集水池可采用敞开式集水池(井)。集水池(井)应设置格栅盖板。
  4. 有可能产生臭气沿盖板周边外溢的污水集水池的检修孔或人孔盖板应密闭。
  5. 集水池除满足有效容积外,还应满足水泵设置、水位控制器、格栅等安装检修要求。
  6. 集水池设计最低水位,应满足水泵吸水要求。当采用潜水排污泵且为连续运行时,停泵水位应保证电动机被水淹没1/2。

建筑排水设计

基本概念

地漏地面标高低于地面5~10mm。

注意地漏、清扫口、检查口。己字弯：5~6层设置一个，用来消能。苏维托混合器：利用横支管与立管连接处的特殊构造，发生溅水现象，使下落水流与空气混合，形成密度小的水沫状汽水混合物，减小下降速度。

两个45°的水里条件要优于一个90人°的条件。

化粪池（卫生间）、隔油池（厨房）、降温池（锅炉、水加热器设备排水）、隔油沉砂池（洗车水）。实验室有毒有害废水、医院污水（含大量致病菌、放射性元素污水）——局部水处理构筑物。

降温池：建筑物附属的发热设备和加热设备排污水及工业废水污水排入城镇下水道水温最高允许值35℃，超过该数据要采取降温处理。

降温方式：二次蒸发、水面散热、加冷水降温。

卫生条件要求高的项目（宾馆、酒店），可以将大便器单独设置立管，防止压强过大破坏洗脸盆、地漏处的水封。

排水立管设计原则：尽量靠近建筑外墙，做不到时考虑靠近客厅，尽量避免卧室，同时尽量先接近大便器。

通气管道：

设置原因：建筑内部排水管道是水气两相流，为防止因气压波动造成的水封破坏,使有毒有害气体进入室内,需设置通气系统。

通气管道：使排水系统内空气流通,压力稳定,防水封破坏而设置的与大气相通的管道。

通气管最小管径不得小于排水管的$\frac{1}{2}$；当排水立管大于50M的时候，通气管管径宜等于通气管。

  • 通气管道遇到必须布置横管的时候，也需要设置一定的坡度。
  • 通气立管汇合管段$DN’=DN + DN×0.25$、$DN’=DN_1+(DN_1+DN_2)×0.25$ （三通气管段连接）。

建筑物内排水流动特点

​ 建筑物内排水系统是水、气、固三种介质的复杂运动。其中固体物少，可以简化为气水两相流。其特点为:断续的非均匀流、气水两相流、管内压力变化。

水流流动状态：附壁螺旋流、水膜流、水塞流（水量从小到大）。

排水管道水力计算

目的：确定管径、坡度、是否需要设立专用排气管。

废水管、污水管同时设计时，应把通气管道布置在两者之间，共用排气。

设计秒流量

（一）、住宅、宿舍（ Ⅰ、Ⅱ类）、旅馆、宾馆、酒店式公寓、医院、疗养院、幼儿园、养老院、办公楼、商场、图书馆、书店、客运中心、航站楼、会展中心、中小学教学楼、食堂或营业餐厅等。

建筑生活排水管道设计秒流量：$q_p=0.12\alpha\sqrt {N_p}+q_{max}$

式中：

  • $q_p$ —— 计算管段排水设计秒流量（L/s）
  • $N_p$ —— 计算管段的卫生器具排水当量总数按表4. 4. 4确定；
  • $\alpha$ —— 根据建筑物用途而定的系数，按表4.4.5确定；
  • $q_{max}$ —— 计算管段上最大的一个卫生器具的排水流量（L/s）， 按表4.4.1取值。

《建筑给水排水设计规范》GB 50015-2003（2009年版）（使用Google Chrome 打开，使用Ctrl+F搜索）

淋浴器与浴盆同时设计时，按浴盆考虑即可，因为两者不会同时用。

排水设计原则：尽量大，不要小。

（二）、宿舍（Ⅲ、Ⅳ类）、工业企业生活间、公共浴室、洗衣房、职工食堂或营业餐厅的厨房、实验室、影剧院、体育场馆等。

建筑的生活排水管道设计秒流量：$q_p=\sum q_0n_0 b$

式中：

  • $q_p$ —— 计算管段排水设计秒流量（L/s）；

新型冠状病毒污染的医疗污水应急处理技术方案

[scode type=”blue”]
转载自：市政给排水交流
[/scode]

背景

深圳市第三人民医院召开记者会，公布最新发现。该院肝病研究所研究发现，在某些新型冠状病毒感染的肺炎确诊患者的粪便中检测出2019-nCoV核酸（新型冠状病毒）阳性，很有可能提示粪便中有活病毒存在。

生态环境部2月1日印发《关于做好新型冠状病毒感染的肺炎疫情医疗污水和城镇污水监管工作的通知》及《新型冠状病毒污染的医疗污水应急处理技术方案（试行）》，安排部署医疗污水和城镇污水监管工作，规范医疗污水应急处理、杀菌消毒要求，防止新型冠状病毒通过粪便和污水扩散传播。

关于做好新型冠状病毒感染的肺炎疫情医疗污水和城镇污水监管工作的通知：

各省、自治区、直辖市生态环境厅(局),新生产建设兵团生态环境局：

为有效应对新型冠状病毒感染的肺炎疫情(以下简称疫情),进一步加强医疗污水和城镇污水监管工作,防止新型冠状病毒通过污水传播扩散,现将有关事项通知如下：

ー、高度重视医疗污水和城镇污水监管工作,将其作为疫情防控工作的一项重要内容抓紧抓实。进一步加强医疗污水收集污染治理设施运行、污染物排放等监督管理;主动加强与卫生健康、城镇排水等部门的协调配合,健全联动机制,形成工作合力。

二、已发生疫情的地方,当地生态环境部门要指导督促接收新型冠状病毒感染的肺炎忠者或疑似患者诊疗的定点医疗机构(医院、卫生院等)、相关临时隔离场所及研究机构,严格行《医疗机构水污染物排放标准》(GB18466-2005),参照《医院污水处理技术指南》(环发(2003)197号)、《医院污水处理工程技术规范》(H)2029-2013)和《新型冠状病毒污染的医疗污水应急处理技术方案(试行)》(见附件)等有关要求,对污水和废弃物进行分类收集和处理,确保稳定达标排放。

对没有医疗污水处理设施或污水处理能力未达到相关要求的医院,应督促其参照《医院污水处理工程技术规范》及《医院污水处理技术指南》,因地制宜建设临时性污水处理罐(箱),采取加氮、过氧乙酸等措施进行杀菌消毒。切实加强对医疗污水消毒情况的监督检查,严禁未经消毒处理或处理未达标的医疗污水排放。对隔离区要指导其对外排粪便和污水进行必要的杀菌消毒。

地方生态环境部门要督促城镇污水处理厂切实加强消毒工作,结合实际,采取投加加消毒剂或臭氧、紫外线消毒等措施,确保出水粪大肠菌群数指标达到《城镇污水处理厂污染物排放标准》(GB18918-2002)要求。

当前公共场所和家庭为防控疫情多采用含氮消毒剂进行消毒,排入城镇污水处理厂的污水余氯量可能偏高,影响生化处理单元正常运行。地方生态环境部门要督促各城镇污水处理厂密切关注进水水质余氯指标的变化情况,及时采取有针对性的应对措施,确保出水达标。

三、未发生疫情的地方,当地生态环境部门要密切关注疫情发展,指导督促本行政区域内医疗机构、接纳医疗污水的城镇污水处理机构等提前做好应对准备。

四、加大农村医疗污水处置的监管力度,指导督促卫生院(所)因地制宜采取加氯、过氧乙酸等措施进行专门的灭菌消毒,防止病毒通过医疗污水扩散。严格污水灌溵的环境管理,禁止向农田灌溉渠道排放医疗污水。

五、进一步加强饮用水水源地保护,做好水质监测,确保饮用水水源不受污染。加大对农贸市场、集贸市场、超市、车站、机场、码头等重点场所污水收集处理的现场监督检査力度,依法查处违法排污,严防发生污染事故。

六、在当地党委政府统一领导下,做好本行政区域内医疗污水和城镇污水处理、污染物排放信息发布工作。按照生态环境部调度安排及时准确统计报送当地医疗污水和城镇污水监管情况、要加与卫生健康、城镇排水、农业农村、公安等部门信息共享强化联防联控,严防疫情扩散蔓延,合力打赢疫情防控阻击战。

新型冠状病毒污染的医疗污水应急处理技术方案(试行)

为了有效应对目前我国发生的新型冠状病毒感染的肺炎疫情(以下简称疫情)患者及治疗过程产生污水对环境的污染,规范医疗污水应急处理、杀菌消毒要求,保护生态环境和人体健康,特制定本方案。

本方案适用于接收新型冠状病毒感染的肺炎思者(以下简称肺炎患者)或疑似肺炎患者诊疗的定点医疗机构(医院、卫生院等)、相关临时隔离场所以及研究机构等产生污水的处理。疫情期间,以上机构产生的污水应作为传染病医疗机构污水进行管控,强化杀菌消毒,确保出水粪大肠菌群数等各项指标达到《医疗机构水污染物排放标准》的相关要求。地方有更严格的地方污染物排放标准的,从其规定。

一、编制依据

  2. (二)《中华人民共和国传染病防治法》

二、总体要求

(一)加强分类管理,严防污染扩散

接收肺炎患者或疑似患者诊疗的定点医疗机构(医院、卫生院等)以及相关单位产生的污水应加强杀菌消毒。对于已建设污水处理设施的,应强化工艺控制和运行管理,采取有效措施,确保达标排放;

对于未建设污水处理设施的,应参照《医院污水处理技术指南》《医院污水处理工程技术规范》等,因地制宜建设临时性污水处理罐(箱),禁止污水直接排放或处理未达标排放。不得将固体传染性废物、各种化学废液弃置和倾倒排入下水道。

(二)强化消毒灭菌,控制病毒扩散

对于产生的污水最有效的消毒方法是投加消毒剂。目前消毒剂主要以强氧化剂为主,这些消毒剂的来源主要可分为两类一类是化学药剂,另一类是产生消毒剂的设备。

应根据不同情形选择适用的消毒剂种类和消毒方式,保证达到消毒效果。

三、采用化学药剂的消毒处理应急方案

(一)常用药剂

医院污水消毒常采用含氯消毒剂(如次酸钠、源白粉、漂白精、液氯等)消毒、过氧化物类消毒剂消毒(如过氧乙酸等)、臭氧消毒等措施。

(二)药剂配制

所有化学药剂的配制均要求用塑料容器和塑料工具。

(三)投药技术

采用含氯消毒剂消毒应遵守《室外排水设计规范》要求。投放液氯用真空加氯机,并将投氯管出口淹没在污水中,且应遵守《氯气安全规程》要求;二氧化氯用二氧化氧发生器;次氯酸钠用发生器或液体药剂;臭氧用臭氧发生器。

加药设备至少为2套,1用1备。没有条件时,也可以在污水入口处直接投加。各医院污水处理可根据实际情况优化消毒剂的投加点或投加量。

采用含氯消毒剂消毒且医院污水排至地表水体时,应采取脱氮措施。采用臭氧消毒时,在工艺末端必须设置尾气处理装置,反应后排出的臭氧尾气必须经过分解破坏,达到排放标准。

四、采用专用设备的消毒处理应急方案

(一)污水量测算

国内市场上可提供的成套消毒剂制备设备主要是二氧化氯发生器和臭氧发生器,这些设备基本可以采用自动化操作方式,设备选型根据产生的污水量而定。污水量的计算方法包括按用水量计算法、按日均污水量和变化系数计算法等,计算公式和参数选择参照《医院污水处理工程技术规范》执行。

(二)消毒剂投加量

1.消毒剂消毒

接收肺炎患者或疑似患者诊疗的定点医疗机构(医院、卫生院等)以及相关单位,采用液氯、二氧化氯、氯酸钠、漂白粉或漂白精消毒时,参考有效氯投加量为50mg/L。消毒接触池的接触时间≥1.5小时,余氯量大于6.5mg/L(以游离氯计),大肠菌群数<100个/L。若因现有氯化消毒设施能力限制难以达到前述接触时间要求,接触时间为1.0小时的,余氯大于10mg/L(以游离氯计),参考有效氮投加量为80mg/L,羹大肠菌群数<100个/L；若接触时间不足1.0小时的,投氯量与余氮还需适当加大。

2.臭氧消毒

采用臭氧消毒,污水悬浮物浓度应小于20mg/L,接触时间大于0.5小时,投加量大于50mg/L,大肠菌群去除率不小于99.99%,粪大肠菌群数<100个/L。

肺炎患者排泄物及污物消毒方法应按照《疫源地消毒总则》相关要求消毒。

关于减压阀：

减压阀阀前应保持水压稳定，管道不宜兼做配水管（即该管道不在接出支管供配水点使用），为了保证供水安全性也可以采用两个减压阀会为备用，减压阀并联设置的作用只是为了当一个阀失效时，将其关闭检修，使管路不需停水检修。减压阀若设旁通管，因旁通管上的阀门渗漏会导致减压阀减压作用失效，故不得设置旁通管。减压阀串联时不得大于两个。可调式减压阀宜水平安装，比例式减压阀宜垂直安装。阀后压力允许波动时，宜采用比例式减压阀 阀后压力要求稳定时，宜采用可调式减压阀。

Tips：

可调式减压阀是出口压力可以在一定范围连续调整的减压阀，可以根据现场的需求，通过转动调节螺钉适当升高或降低阀后压力，并保持阀后压力稳定。满足工艺设备不同压力的需求。适用范围广，调整方便。常用减压阀型号Y42X薄膜减压阀，Y43H先导活塞减压阀，YD43H先导薄膜减压阀，PD17先导薄膜减压阀，YK43X先导活塞减压阀，200X先导薄膜减压阀，Y44H波纹管减压阀，Y14H波纹管减压阀等都是可调式减压阀，通过转动调节螺钉和更换不同弹簧可以调整出口压力变化。大多数的减压阀都是可调式减压阀。
比例式减压阀的进口压力和出口压力始终保持一定的比值，如2:1,3:1,3:2等。当减压阀进口压力上升时，减压阀出口压力也会上升。当减压阀进口压力下降时，减压阀出口压力也会下降。始终保持固定的比例值。适用于对阀后压力变化要求不严格的场合。比例式减压阀压力比值是通过活塞式阀芯两端直径比值决定的。加工装配好后其进出口压力比值就固定了，无法更改。所以可调式减压阀的出口压力在一定范围可调，并能够保持阀后压力稳定不变。比例式减压阀的出口压力是随着进口压力的变化而同时变化，并且阀后压力不可调整，只是同阀前压力保持固定比值。

止回阀，单向阀和倒流防止器的区别

止回阀，是单向阀，自动防止流体反向流动，无旋钮，不能人为阻断正向流动。

截止阀，有旋钮，能人为控制正向流动，由于其特殊结构，有变流量的可能，有双向关闭的应用。

倒流防止器产品由两个隔开的止回阀和一个液压传动的泄水阀组成。

由于止回阀的局部水头损失，中间腔内的压力始终低于入水口的压力。

这个压差驱使泄水阀处于关闭状态，管路正常供水。

在压力异常时，（即出口端压力高于中间腔），即使两个止回阀都不能反向密封，安全泄水阀也能自动开启将倒流水泄空，并形成空气隔断，保证上游供水卫生安全。

倒流防止器 是一种严格限定管道中的压力水只能单向流动的水力控制组合装置。

SWMM设计软件技术简介

SWMM( storm water management model,暴雨洪水管理模型)是一个动态的降水-径流模拟模型,主要用于模拟城市某一单一降水事件或长期的水量和水质模拟。其径流模块部分综合处理各子流城所发生的降水,径流和污染负荷。其汇流模块部分则通过管网、渠道、蓄水和处理设施、水泵、调节闸等进行水量传输。该模型可以跟踪模拟不同时间步长任意时刻每个子流城所产生径流的水质和水量,以及每个管道和河道中水的流量、水深及水质等情况。

SWMM自1971年开发以来,已经经历过多次升级。在世界范围内广泛应用于城市地区的暴雨洪水、合流式下水道、排污管道以及其它排水系统的规划、分析和设计,在其它非城市区域也有广泛的应用。

SWMM5提供了一个宽松的综合性环境,可以对研究区输入的数据进行编辑、模拟水文水力初水质情況,并可以用多种形式对结果进行显示，包括对排水区域和系统输水路线进行彩色编码,提供结果的时间序列曲线和图表、坡面图以及统计频率的分析结果。

地勘报告的识读

对于地勘报告,全部阋读是没有必要的。一般应有侧重的阅读：

  • 第一,先看结语和建议中的持力层土质、地基承载力特征值、地基类型以及基础处理的建议；
  • 第二,结合钻探点看懂地质剖面图,进步确定设计管道基础所在的位置；
  • 第三,岩土工程分析预评价中看是否存在饱和沙土地基,是否有液化；
  • 第四,重点看下地基处理方案与设计施工建议章节的内容；
  • 第五,特别扫读一下施工期间和后期运行可能发生的岩土工程问题及预防措施的建议,必要是把它加到设计说明有关基础处理的具体描述中。

管道沟槽开挖后的地基验槽

给排水管道施工作业作为埋地隐蔽工程,在施工环节中,验槽是在管道基槽开挖后保证管道施工质量的重要工序,也是一般岩土工程勘察工作的最后个环节。

基槽全面开挖后,地基持力层土层会完全暴露出来,首先检验勘察成果与现场实际情况是否一致，如有不相符的情况,应协商解決,修改设计方案,或时地基进行处理等措施。

验槽环节应着重查明下列情况：

  1. 基槽开挖后,地质情况与原提供地质报告是否相符。
  2. 场地内是否有新近沉积土,边坡是否稳定。
  3. 是否有因雨、雪、天寒等情况使基底岩土的性质发生了变化。
  4. 场地内是否有被扰动的岩土。
  5. 地基基础应尽量避兔在雨季施工。无法避开时,应采取必要的措施防止地面水和雨水进入槽内,槽内水应及时排出,使基槽保持无水状态,水浸部分应全部清除。
  6. 严禁局部超挖后用虚土回填。

管道地基处理的目的

管道的设计、施工、维护等有它的特殊性,和地形、地质、输送的介质、管材等有着密切的关系。管道安装中采用直埋的方法最为普遍,而直埋管道的基础对不同地基、土质也有着不同的要求。合格管材与良好的施工质量是建设优质管道工程必不可少的,其中要特別注意在不良土质情况下管道基础的设计及施工。

不良土质般是指天然含水量大、压缩性高、承载能力低、天然孔隙比大于或等于1.0的一种软塑到流塑状态的粘性软土。如淤泥、淤泥质土、泥炭、泥炭质土等;其承载力较低,承受荷载后沉降変形较大。以软土作为地基持力层,如果地基处理不当,很容易由于过度沉降成局部不均匀沉降而导致管道及接口处因变形过大引起开裂,导致管道渗漏。滲漏的恶性发展反过来又进一步加大了管道基础的变形破坏。

影响管道地基的因素包括:地基土强度及稳定性、压缩変形及不均匀沉降、流砂以及地震或外力影响。

在不良土质地区的管道工程发生事故的主要原因是地基的不均匀沉降，因此管道工程对地基强度、稳定性及不均匀沉降有极为严格的要求。

管道基础的形式大致可以分为混凝土基础、天然土基础和砂石基础三类,当管底原状土地基承载力较低,而管道本身对抵御不均匀沉降的要求较高时,宜采用混凝土基础。

当埋地金属管道的地基持力层土质条件较好、无地下水目地面活荷载较小时,可以考虑采用天然土弧基础。

当天然土弧基础不能满足与管底孤度吻合,而地基承载力较高时,宜采用砂石基础以适应管底外形。

管道地基处理的方法

地基处理是人为改善岩土的工程性质或地基组成,使之适应基础承载力需要而采取的措施。

所谓基础,是指将上部结构所承受的各种作用传递到地基上的结构组成部分,根据埋置深度不同可分为浅基础和深基础。浅基础包括独立基础(含刚性基础和扩展基础)、条形基础、十字交叉基础、筏板基础等;深基础包括桩基础、沉井基础、墩基础、箱形基础及地下连续墙等。

给水管道工程的基础形式般为浅基础,因长直管道的基础往往连续布设,故作为条形基础进行设计是合理的。给水管道所经场地因地基土性状不ー,并非所有天然地基都能直接铺设管道,这就涉及到地基处理问题。

常用地基处理方法有:置换法,排水固结法,压实和夯实法,振密和挤密法,加筋法等。

置换法中的换土垫层法和褥垫法在给排水管道工程中使用较多。

管道地基处理的常用方法

褥垫法：是复合地基中解决地基不均匀的一种方法。如当管道一边位于岩石地基,而边位于黏土地基时,为确保两者之间变形的协调,可采用在岩基上加褥垫层(级配砂石)的方法来解决。蘅垫层厚度可取200~300mm,其材料可选用中粗砂、级配砂石等,最大粒径不宜大于20mm。

深层搅拌桩法：是用深层搅拌机械把水泥浆液送入地下,通过和原位地基土强制混合,使原来的软土地基变成具有整体性、水稳定性和一定强度的加固土桩复合地基,从而提高地基承载力,减少地基沉降,达到加固地基的目的。

换填垫层法：是将基础下一定深度范围的软弱土层全部或部分挖除,然后分层回填并夯实砂、碎石、素土、灰土等强度较大、性能稳定和无侵蚀性的材料,并夯实至设计要求的密实度,从而提高浅层地基承载力,减小地基沉降量。换土垫层法主要用于淤泥、淤泥质土、杂填土等软弱地基的浅层处理。垫层的压实标准用压实系数(度)来衡量,一般不小于0.94。

管材的选用

管材选用的一般要求

  1. 管材、管件应符合国家现行产品标准,并宜采用同生产厂家的相应配套产品；
  2. 管材、管件公称压力(包括管道接口耐压)应与管道工作压力匹配,管道内工作压力不得大于产品标标定的公称压力或标称的允许工作压力；
  3. 管材应具有足够的强度和水密性,可以承受各种外部荷载；
  4. 管材内壁尽量光滑,以减少水头损失和敷设坡度；
  5. 管材宣重量轻,以运输及施工安装；
  6. 管材宜价格低,耐腐蚀,使用年限较长；
  7. 生活给水管材、管件所涉材料必须符合现行生活饮用水卫生标准；
  8. 排水管材以就地取材,并考虑到预制管件和快速施工的可能。

管道的压力及管径的概念

管材公称压力 nominal pressure

冷水管材在20℃的水温条件下,使用年限为50年,产品标准所给定的最大允许工作压力。

系统工作压力 system operating pressure

管道系统工作时,作用于管内壁的最大持续压力,不包括试验压力和水锤压力。

设计压力是指给水管道系统作用在管内壁上的最大瞬时压力。一般采用工作压力及残余水锤压力之和。

三者的关系：公称压力>=设计压力=1.5×工作压力

一般来说,管子的直径可分为外径、内径、公称直径。公称直径,又称平均外径,是指标准化以后的标准直径,为内径和外径之间的中点。以DN表示,单位mm。

关于管道规格的表示：因管材而有所不同,如无缝钢管的外径用字母D来表示,其后附加外直径的尺寸和壁厚,例如外径为108的无缝钢管,壁厚为5mm,用D108*5表示;塑料管也用外径表示,如De63，其他如钢筋混凝土管用d表示;铸铁管等用DN表示,在设计图纸中一般采用公称直径来表示。

公称直径是为了设计制造和维修的方便人为地规定的一种标准,也叫公称通径,是管道或阀门的规格名称。管子的公称直径和其内径、外径都不相等。在设计图纸中之所以要用公称直径,目的是为了根据公称直径可以确定管子、管件、阀门、法兰、垫片等结构尺寸与连接尺寸,公称直径采用符号DN表示,如果在设计图纸中采用外径表示,也应该作出管道规格对照表,表明某种管道的公称直径,壁厚。

城镇给水管道工程常用管材性能特点

（一）金属管材

在常用管材中,金属管材主要包括两种：、

  • 钢管(又分为无缝钢管和焊接钢管两种)
  • 给水铸铁管(又分为灰铸铁和球器铸铁)

对于钢管来说,目前采用的承插式焊接接口,改用搭接焊缝接口,对传统钢管的对接焊缝接口进行了改良,提高了接口焊缝的质量,有效避兔了管道爆漏。接口采用焊接或法兰接口,所用配件多由钢板卷焊而成。与球墨铸铁管相比,钢管具有韧性好、抗拉强度大、管壁薄、耐高压、管节长、接口少等优点,最大的缺点是耐腐蚀性差、价格高。多应用于室外高水压、穿越河谷铁路及地震地区等管段的大管径给水管。

对于给水铸铁管来说，其具有轻度高、壁薄、耐压、耐冲击、耐腐蚀、耐抗震等性能。管道接口采用承插式和法兰式两种接口方式,且还有一定的延伸率及偏转角。具备生铁管和钢管材质的优点,避兔了铁和钢的缺点。球墨铸铁的抗拉强度是灰铸铁的一倍,屈服强度甚至超过铸钢。所以,球墨铸铁排水管的抗压强度高,埋在地下,不容易被汽车压破。因此多应用于室外大管径给水管。

（二）非金属管材

非金属管材可分为三类：

第一类是预应力钢筋混凝土管,预应力钢筋混凝土管分普通和加套筒两种。预应力套筒钢筋混凝土管（PCCP管）是在预应力钢筋混凝土管内放入钢筒。其接口为承插式,承口环和插口环均用扁钢压制成型,与缸筒焊成体。管件多采用钢管管件。其特点是抗压(可承受压力范国在0.4~1.2MPa)、价格便宜(相对于金属管材而言)、较好的抗渗性、耐久性、防腐性,方便就地取材,从而减少钢材的使用。预应力钢筋混凝土管作压力给水管,可代替钢管和铸铁管,降低工程造价,它是目前我国小城镇常用的给水管材。主要缺点是性脆、怕硇碰撞等；

第二类是玻璃钢管，玻璃钢管也称玻璃纤维缠绔绕夹砂管(RPM管）。主要以玻璃纤维及其制品为增强材料,多采用橡胶圈承插连接、法连接及胶泥连接。具有轻便、耐腐蚀、强度好、水头损失小等优点,方便运输、连接和吊装。缺点是造价较高,刚性低、易损坏,专业性安装和开挖回填的要求高,施工费用较高。

最后一类是塑料管，塑料管材作为化学建材的重要组成部分,以其优越的性能,卫生、环保、低耗等优点为用户所广泛接受,主要有UPVC给水管、聚乙烯（PE）给水管材等。其工艺发展比较完善,都属于城镇给水工程建设常用管材。

塑料管材与传统的金属管和水泥管相比,具有重量轻,一般仅为金属管的1/6-1/10,有较好的耐腐蚀性,抗冲击和抗拉强度,塑料管内表面比铸铁管光滑的多、摩擦系数小、流体阻力小、可降低输水能耗5%以上,综合节能好,制造能耗右降低75%,运输方便,安装简单,使用寿命长达30-50年。

PE管材耐温性好,可在-40℃到60℃范围内使用,冬季安装施工不会发生管道脆裂。与其他常用塑料材料相比,聚乙烯的耐莓菌性能要高很多,长期使用也不积垢。其常用的连接方式有热熔连接、电熔连接、承插式柔性连接、法兰连接、钢塑过渡接头连接等。

（三）给水配件与控件

给水配件：在管线转弯、分支、直径变化处及连接其他附属设备处,需采用各种标准配件。

给水控件：主要有调节流用的阀门、供应消防用水的消防栓，其他的还有控制水流方向的单向阀、安装在管线高处的排气阀和低处的排泥阀。

排水管主要承担雨水、污水、农田排灌等排水的任务。

常用的排水管为塑料排水管和钢筋混凝土管(RCP)。

高密度聚乙烯(HDPE)双壁波纹管以约100%的高密度聚乙烯HDPE为材料,采用特殊膜压工艺，在热熔状态下整体一次膜压成型。适用于管径DN225~DN1000规格的排水管道工程施工,管材长度般为6m。工作温度为-40℃~60℃,环刚度≥8kN/m2。管道连接方式:“O”型橡胶圈承插口连接,渗透率≤2%。具有密度高、重量轻、抗震性好、耐低温、抗冲击强、耐腐蚀,并具有良好的输水性能,使用寿命长,施工进度快,工程综合造价低等优点。

钢筋混凝土管管材使用历史最长,具有较成熟的制作工艺和施工经验,可以根据不同的埋深、内压(压力指数,I级Ⅱ级Ⅲ级)、口径(平口、企口、承插口双插口、钢承口)进行配制,接口多采用橡胶止水带,止水效果较好,价格较低,性能稳定。施工较方便,适用于开槽埋管和管施工。缺点,重量大,管节多,安装及起吊设备要求高。

市政给排水的规划

城市规划的分类

  • 总体规划
  • 详细规划（控制性详细规划、修建性详细规划）

***总体规划:***是指在一定区域内,根据国家社会经济可持续发展的要求和当地自然、经济、社会条件对土地的开发、利用、治理、保护在空间上、时间上所做的总体安排和布局。

***控制性详细规划:***用以控制建设用地性质、使用强度和空间环境的规划。主要应用指标量化、条文规定、图则标定等方式对各控制要素进行定性、定量、定位和定界的控制和引导。是城乡规划主管部门作出规划行政许可、实施规划管理的依据,并指导修建性详细规划的编制。

***修建性详细规划:***是以城市总体规划、分区规划或控制性详细规划为依据,制订用以指导各项建筑和工程设施的设计和施工的规划设计,是城市详细规划的一种。

***专项规划:***是为了更好地实施城市总体规划,是依据城市总体规划,对专项工程规划内容的细化和深化。

市政给排水规划相关的法规

城市规划工作的基本内容是依据城市的经济社会发展目和环境保护的要求,根据区域规划等更高层次的空间规划的要求,在充分研究城市的自然、经济、社会、文化和技术发展条件的基础上,结合城市发展战略,预测城市发展规模,选择城市用地的布局和发展方向,按照工程技术和环境的要求,综合安排城市各项工程设施,并提出近期控制引导措施。

市政工程系统也称市政公用设施或城市公共设施,是城市基础设施的主要内容,主要包括:

  • 水务工程系统-给水、排水
  • 能源工程系统-电力、燃气供热
  • 通信工程系统-电信
  • 防灾工程系统一防洪、抗震、消防

市政工程规划作为城市发展政策性法规和后阶段设计工作指导性文件,具有一定的法律严肃性。

城镇给排水规划,首先应贯彻执行《城市规划法》、《水法》、《环境保护法》、《城镇排水与污水处理条例》、《水污染防治法》等国家相关法规政策,同时应以区域总体规划、城镇总体规划为依据,并应与水资源规划,水污染防治规划、生态环境保护规划和防灾规划等相协调。其次城镇排水规划与城镇给水规划应相互协调。

城市给水工程系统规划的原则

  1. （水资源作为一种特殊的自然资源,其所有权归国家所有）。城市给水系统规划应遭守国家相关的法规文件。
  2. （关于城市城市水量的分配）给水系统规划应优先保障城市生活用水,统筹兼顾、发挥水资源的多种功能;缺水地区应限制城市规模和耗水量大的工业行业发展,以确保水资源的可持续性利用；城市供水水源开发利用规划应与水资源规划协调一致。
  3. （水质防护）城市生活给水系统的供水水质必须符合现行的生活饮用水卫生标准,城市管道的连接敷设、蓄水构筑物的选址和设计应当防止水质的二次污染。对水源的保护必须在给水系统规划中予以体现,同时饮用水水源保护区的设置和水污染防治应纳入当地的社会经济发展规划。
  4. 给水系统的整体布局和工艺流程应满足城市的水量、水质、水压及城市消防、安全供水的要求。
  5. 城市给水系统规划期限应与城市总体规划期限一致,应按远期规划,以近期为主,近远期结合的原则进行设计。

城市给水工程规划的内容

给水工程设施规划是城市规划的重要组成部分，应编制总体规划、分区规划，有时另行编制专业规划。在此基础上，再编制城市给水系统详细规划。

城市给水系统总体规別的内容与深度如下:

  1. 城市给水水源工程规划——估算城市用水总量,根据水源水质及水量情况选择水源；
  2. 城市取水工程规划——确定取水位置及取水方式;同时确定水源地卫生防护措施；
  3. 城市给水处理厂规划——根据水源水质变化情况和供水水质目标选择水处理工艺形式；
  4. 城市给水管网规划——根据城市发展布局规划,选择给水设施的位置输配水干管走向估算干管管径；
  5. 城市节约用水规划和再生水工程——图纸内容有:水源点、泵房、水厂、清水池的平面位置图,输配水千管走向、管径、加压泵站位置图。

城市给水工程详细规划的内容与深度

城市给水工程详细规划作为给水工程设计的主要依据，结合当地实际可对总体规划及分区规划作出评价，其主要内容为：

  1. 确定用水量标准,预测城市用水量,对水资源与城市用水量供需平衡进行分析
  2. 选定城市给水水源和取水点位置
  3. 确定供水水质目标选定自来水厂位置和水处理工艺基本流程
  4. 确定集中供水、分区供水方式确定加压泵站位置、标高、容量
  5. 确定输配水管走向、管径对环状管网进行必要的管网平差及校核
  6. 选择输配水管网管材及敷设方法,提出跨越河道和障碍物的方法
  7. 对整体规划方案进行工程投资估算或匡算预测工程效益
  8. 提出水资源保护以及开源节流的要求和措施
  9. 对总体规划或专业规划作出评价,提出详细规划的修改理由。主要图纸的表达内容为:输配管网的走向管径、埋深给水泵站的位置等。

城市用水量预测方面的修订

城市供水总预则的影响因素:人口的増长;生活条件;用水习惯;资源价值观念;科学用水和节约用水的技术措施;水价和水资源丰富和紧缺程度。且根据其他国家的城市供水规律,用水量増长到一定程度后会达到个稳定水平。

原版规范的用水量指标远大于国外的人均综合用水量,水量预测方法上的分类估算法、年递増率法等计算方法过分偏重理论计算,没有考虑用水现状和节水政策的影响
根据”给水工程规划应从全局出发,坚持保障供给、水资源可持续利用,建设节水型城市的原则“。新版城市给水规划规范在用水量指标和用水量计算方面进行了大的调整。以充分体现规划设计的合理性、科学性和可实施性,优化给排水系统建设的可持续发展目标和综合服务功能。

调整的范围包括：

  • 用水量指标值,参考一些城市建设统计年鉴,指标值均有所下调；
  • 城市规模分档的调整,建城市规模划分为五类七档；
  • 城市用水区域的调整,将原一区的重庆、四川、贵州、云南等调至二区。

市政项目的基本建设工作程序

遵循基本建设程序,先规划研究,后设计施工,有利于加强国家宏观经济计划管理,保持建设规模和国力相适应;也有利于保证项目決策正确,积极地完成建设任务,提高投资效果。

一般市政工程项目的全过程都要经过：

  1. 提出项目建议书；
  2. 进行可行性研究；
  3. 工程设计(从技术和经济上对找拟建工程作出详尽规划)；
  4. 建设准备；
  5. 建设施工；
  6. 竣工验收交付使用等六个阶段。

规划一般属于上设计程序,即先可研，规划，再初设。

Tips：

  1. 规划设计规划设计院等规划编制单位有规划编制资质的可以做,资质可分甲、乙、丙级；
  2. 可研，咨询机构，或者有咨询资质的；
  3. 初步设计，施工图设计，有工程设计资质，也有些单位是综合性的，可以都同时做。

项目立项

大中型市政工程项目经过项目实施组织決策者和政府有关部门的批准,并列入项目实施组织或者政府计划的过程叫项目立项。报批程序结束即为项目立项完成。立项分类鼓励关、许可类、限制类，分别对应的报批程序为备案制、核准制、审批制。申请项目的立项时,应将立项文件递交给项目的有关审批部门。立项报告包括项目实施前所涉及的各种由文字、图纸、图片、表格、电子数据组成的材料。不同项目、不同的审批部门、不同的审批程序所要求的立项文件是各有不同的。

项目立项是国有企业或政府投资项目单位为推动某个项目上马,根据国民经济的发展、国家和地方中长期规划、产业政策、生产力布局、国内外市场、所在地的内外部条件,提出的具体项目的建议文件,是专门对拟建项目提出的框架性的总体设想,该报告的核心价值是：

  • 作为项目拟建主体上报审批部门审批决策的依据的依据；
  • 作为项目批复后编制项目可行性研究报告的依据；
  • 作为项目的投资设想变为现实的投资建议的依据；
  • 作为项目发展周期初始阶段基本情况汇总的依据。

基本建设程序是基本建设项目从决策、设计、施工到峻工验收整个工作过程中各个阶段必须遭循的先后次序。国家规定的基本建设程序不可逾越,只有上一环节的工作获批准后方可转入下一工作环节。

各设计阶段主要内容及深度要求：

1、方案设计

方案设计(概念设计)是投资决策之后,由咨询单位将可行性研究提出意见和问题,经与业主协商认可后提出的具体开展建设的设计文件、其深度应当满足编制初步设计文件和控制概算的需要。

2、初步设计

初步设计(基础设计)的内容依项目的类型不同而有所变化,一般来说,它是项目的宏观设计、即项目总体设计、布局设计、主要的工艺流程、设备的选型和安装设计、土建工程盐及费用的估算等。初步设计文件应当满足编制施工招标文件、主要设备材料订货和编制施工图设计文件的需要,是下一阶段施工图设计的基础。

3、施工图设计

施工图设计(详细设计)的主要内容是根据批准的初步设计,绘制出正确、完整和尽可能详细的建筑、安装图纸包括建设项目部分工程的详图、零部件结构明细表、验收标准、方法、施工图预算等。此设计文件应当满足设备材料采购、非标准设备制作和施工的需要,并注明建筑工程合理使用年限。

规划编制及前期工作的目的程序

编制项目规划的主要目的是有效地指导各类城市建设活动,宏观层面上把握工程系统设施发展的定性、定位、定向。以及生态环境保护等。拟定规划期內具体工程系统设施在规划区范围内的发展所应达到的目的和指标。保证各类建设工程有序地进行建设;规划的编制过程一般可分为4个阶段：

  • 规划启动阶段
  • 规划调硏准备阶段
  • 规划编制阶段
  • 规划收尾阶段

项目建议书和可行性研究报告属于项目前期准备阶段,其目的是争取项目立项。初步设计或实施方案属于项目实施阶段，是项目立项后的一项基础工作。

规划編制过程中前期分析阶段的工作内容

编制城市总体规划前,应对当地现行城市总体规划以及各专项规划的实施情况进行总结,对基础设施的支撑能力和建设条件做出评价;针对存在问题和出现的新情况,从土地、水、能源和环境等城市长期的发展保障出发,依据有关规划,着眼区域统筹和城乡统筹,对城市的定位、发展目标、城市
功能和空间布局等战略问题进行前瞻性研究,作为城市总体规划编制的工作基础。

编制前应做好调查和资料收集：

  • 现场踏勘或观察调查应收集基础资料的内容：现场踏勘(城市概貌、新发展地区、原有地区、重要工程)、基础资料的收集与整理;(基础资料有城市自然条件及历史资料、技术经济资料、现有工程设施及环境资料等)、分析研究(系统分析整理,定性到定量，提出解決问题的对策)。
  • 抽样调查或可卷调查。
  • 访谈和座谈会调查。
  • 文献资料的运用及类比。

城市视划的三个阶段

市政工程规划的三个阶段各规划层次阶段的任务和主要内容如下：

(1)总体规划阶段的任务和内容

综合协调并确定城市供水、排水、防洪、供电、通信、燃气、供热、消防、环卫等设施的总体布局。以规划图表示。标明给水排水、热力、煤气、电力、电信等主要管线的走向,水源、水厂、热电站、污水处理厂、气源、调压站、电厂、变电站等主要构筑物位置。

(2)分区规別阶段的任务和内容

在总体规划基础上,对城市土地利用、人口分布和公共设施、城市基础设施的位置作进步的安排。规划内容为:确定工程管的位置、走向、管径、服务范围以及主要工程设施的位置和用地范围。

(3)详细规划阶段的任务和内容

详细规定建设用地的各项控制指标和其他规划管理要求。规划内容有:各种管线的平面和竖向规划设计;估算工程量、拆迁量和总造价、分析投资效益。

市政给排水项目规划工作的原则

城市雨水系统的概念

城市雨水系统:收集、输送、调蕾、处置城市雨水的设施及行洪通道以一定方式组合成的总体,包括：

雨水排放系统：应对常见降雨径流的排水设施以一定方式组合的总体,以地下管网系统为主。亦称“小排水系统”。

源头减排系统：场地开发过程中用于维持场地开发前水文特征的生态设施以一定方式组合成的总体。

防涝系统：应对内涝防治设计重现期以内的超出雨水排放系统应对能力的强降雨径流的排水设施以一定方式组合成的总体。亦称“大排水系统”。

城镇内涝防治措施：是指通过源头控制、排水管网完善、城镇涝水行泄通道建设和优化运行管理等综合措施防治城镇内涝。包括工程性措施和非工程性措施。

雨水综合管理：是指通过源头削减、过程控制、未端处理的方法控制面源污染、防治内涝灾害、提高雨水利用程度。

面源污染：是指通过降雨和地表径流中刷,将大气和地表中的污染物排入受纳水体,使受纳水体遭受污染的现象。

雨水利用：雨水入渗、收集回用、调蓄排放等的总称。

调蓄排放设施：储存一定时间的雨水,削减向下游排放的雨水洪峰径流量、延长排放时间的设施。

城市雨水系统的排放控制模式

根据城镇雨水系统的作用和功能,可分为以下类型：（专项规划也分为相应类型）

  • 传统的雨水排水系统：以快速收集、输送、排除雨水为目的的雨水排水系统，以保障城镇不受降雨积水的影响。
  • 雨水径流量控制系统：通过采用一系列技术措施,降低局部区域或城镇雨水径流量，减少低洼区域的积水,防治或减轻城镇内涝。
  • 雨水径流污染控制系统：通过采取一些列技术惜施,对初期雨水截留、处理,以减轻初期雨水排入环境的污染负荷。
  • 雨水综合利用系统：对雨水资源收集、处理利用的系统。

内涝：强降雨或连续性降雨超过城镇排水能力,导致城镇地面产生积水灾害的现象。(积水深度、积水时间和范围)

内涝防治系统：用于防止和应对城镇内涝的工程性设施和非工程性措施以一定方式组合成的总体,包括雨水收集、输送、调蓄、行泄、处理和利用的天然和人工设施以及管理措施等。

编制城镇内涝防治规划的政策背景

国家出台的《城镇排水与污水处理条例》于2013年10月2日公布,自2014年1月1日起施行。

《城镇排水与污水处理条例》要求易发生内涝的城镇、编制城镇内涝防治专项划,纳入本行政区域的城镇排水与污水处理规划。针对城市内涝灾害问题,条例规定了一系列制度措施：

  • 一是从规划层面作出规定,要求易发生内满的城市、镇编制城镇内淓防治专项规划,纳入本行政区域的城垅镇排水与污水处理规划。城镇内淓防治专项规划的编制,应当根据城镇人口与规模、降雨规律、暴雨内涝风险等因素,合理确定内涝防治目标和要求,充分利用自然生态,提高雨水滞渗、调蕾和排放能力。
  • 二是规定市政基础设施工程应当配套建设雨水收集利用设施,増加绿地、可滲透路面等对雨水的滞渗能力;新区建设与旧城区改建,应当按照雨水径流控制要求建设相关设施。
  • 三是规定地方政府组织有关部门、单位编制应急预案,建立内涝防治预、会商、联动机制,统筹安排排涝物资,加强易涝点治理,共同做好内淓昉治工作;加强设施建设和改造,发挥河道行洪能力。采取清淤疏浚措施,确保排水畅通;在汛期,防汛指挥机构要加强对易涝点的巡查,及时排除险情。
